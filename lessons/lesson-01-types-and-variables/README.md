# Lesson 1: Types & Variables

Every value in a program has a type. The type tells the computer (and you) what kind of thing it is and what you can do with it. This lesson is about recognizing types in code you've already seen and writing your own.

## What you'll learn

- What a type is and why languages have them
- The most common C# types: `int`, `float`, `string`, `bool`
- How to declare and use variables
- The difference between `var` and an explicit type name
- How constants work
- Where all of this lives inside the mod you already built

## Success criteria

You're done with this lesson when you can:

- [ ] Name at least four primitive types in C# and give a real-world example of each
- [ ] Explain the difference between `var goldValue = 10;` and `int goldValue = 10;`
- [ ] Explain what `const` does and why you'd use it
- [ ] Point to at least three variables in `CoinFlip.cs` and name their types
- [ ] Build the mod successfully after making your own change
