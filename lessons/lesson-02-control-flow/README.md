# Lesson 2: Control Flow

Programs rarely run straight from top to bottom. Most of the interesting behavior in code comes from decisions: do this *or* that, do this *only if* some condition is met. That's control flow.

This lesson covers the tools C# gives you to make decisions: conditions, `if`/`else`, and `switch`. By the end you'll be able to read every branching line in the starter mod — and add a new branch of your own.

## What you'll learn

- How to write a condition (a true/false expression)
- Comparison operators and logical operators
- The anatomy of `if`, `else if`, and `else`
- The ternary operator `? :`
- Switch statements and the modern switch expression
- When to reach for each tool

You're done with this lesson when you can:

- [ ] Write a boolean expression using at least two logical operators (`&&`, `||`, `!`)
- [ ] Write an `if`/`else if`/`else` chain with comparison operators
